Step 1 – Install Tailscale

Tailscale is a secure VPN that lets your phone connect to the AquaScan demo server over a private network.

  • Install Tailscale from the Play Store (search “Tailscale”).
  • Open the app and choose “Sign in with Google”.
  • Sign in using the demo account:
    Email: demoaquascan@gmail.com
    Password: aquascan2025
  • Once logged in, ensure it shows “Connected”.

This is a shared demo account for the exhibition. Please do not change the password.

2

Step 2 – Ensure You’re Connected

After signing into Tailscale with the demo account, your device joins the same private network as the AquaScan server.

  • Keep Tailscale running in the background.
  • No extra configuration needed — the app uses this connection automatically.
3

Step 3 – Download AquaScan (Android APK)

AquaScan isn’t on the Play Store yet. For this demo, install it via an APK file.

Download AquaScan APK

After download, tap the file and allow installation from unknown sources if prompted.

4

Step 4 – Open AquaScan & Scan a Fish

  • Make sure Tailscale is still connected.
  • Open the AquaScan app.
  • Tap “Scan from camera” or “Scan from gallery”.
  • The app sends the fish image to the local server and shows:
    • Predicted disease class
    • Confidence
    • History of previous scans
  • Use the AquaScan AI Assistant to ask questions about fish diseases.

Note: This demo only works while the AquaScan server is running on the local network during the exhibition.

Behind the scenes

  • AquaScan uses a fish disease classification model running on a local Python server.
  • The AI assistant uses an OpenAI model (GPT-4o Mini) via a secure workflow in n8n.
  • All traffic stays private through Tailscale (no public cloud server).
